# 二、区分用户点击事件和 JS 触发的点击事件

Event 接口的 isTrusted 属性是一个只读属性，它是一个布尔值（Boolean）。当事件是由用户行为生成的时候，这个属性的值为 true ，而当事件是由脚本创建、修改、通过 EventTarget.dispatchEvent() 派发的时候，这个属性的值为 false 。

if (e.isTrusted) {
  /* Event 事件可信 */
} else {
  /* Event 事件不可信 */
}
