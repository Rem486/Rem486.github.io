# vuex module 使用 rootState

在 vuex module 里面使用根节点状态的方法：

const moduleA = {
  actions: {
    add ({ state, commit, rootState }){
      commit('ADD', state.count + rootState.count);
    }
  },
  getter: {
    total (state, getters, rootState){
      return state.count + rootState.count;
    }
  }
}

# label 标签会改变 event target

HTML label 元素（标签）表示用户界面中某个元素的说明。 不要在 label 元素内部放置可交互的元素，比如 anchors 或 buttons。这样做会让用户更难激活/触发与 label 相关联的表单输入元素。

一般是使用 label 标签可以和 input 相关联，来扩大 input 的点击区域。这也就导致了，其内部的点击事件都会被关联到 input 上面去，event target 都是 input。导致一些判断出错。
