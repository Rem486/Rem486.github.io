# 操作方法

1.先修改 hosts 文件

Windows: C:\Windows\System32\drivers\etc\hosts

Linux/Mac: /etc/hosts

2.hosts 文件增加一条，放在最后吧

# Jetbrains
0.0.0.0 account.jetbrains.com
1
2

3.添加激活码

一般都会推荐先断网比较好。

打开要激活的产品，这里 WebStorm 为栗。

菜单栏 > Help > Register

激活方式选择 Activation code，输入激活码。点 OK 完事。

# 亲测可用

更新时间 2020/02/11

使用 破解补丁+注册码 的方式，激活成功。

# 1. 下载 WebStorm 2019.3 版本，试了 WebStorm 2019.3.2 好像不行。

官网下载历史版本

# 2. 删除老版本后，安装。

# 3. 选择试用30天。

# 4. 配置破解补丁。

下载破解补丁，把 jetbrains-agent.jar 文件放置在安装目录下的 bin 目录里面。

打开 WebStorm，点击菜单栏 【Help】 -> 【Edit Custom VM Options】。在文件里面配置破解补丁的位置。 注释：Mac 和 Windows 的路径可能不一样。

# windows
-javaagent:C:\Program Files\JetBrains\WebStorm 2019.3\bin\jetbrains-agent.jar
# Mac
-javaagent:/Applications/CLion.app/Contents/bin/jetbrains-agent.jar
1
2
3
4

重启 WebStorm

# 5. 填入注册码

点击菜单栏【Help】 -> 【Register】，粘贴注册码，Activate。

成功激活到 2089年，🎉 🎉
