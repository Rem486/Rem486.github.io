# 声明式和命令式

创建时间：2019-08-12 ###遇到问题 傻傻分不清

  • 命令式

程序花费大量代码来描述用来达成期望结果的特定步骤 —— 控制流：即如何做。

  • 声明式

程序抽象了控制流过程，花费大量代码描述的是数据流：即做什么。
