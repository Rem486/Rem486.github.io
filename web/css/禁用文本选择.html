# 禁用文本选择

创建时间：2019-09-29

# 遇到的问题

需要做滑块拖动，但是拖动的时候，老是容易选中旁边的文本，出现蓝色底纹的选中效果。就可以禁用相关文字被选中。

# 解决方案

给对应元素加上 user-select: none;

p{
   user-select: none;
}
