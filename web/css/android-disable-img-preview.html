# 禁用安卓点击图片预览

img {
    pointer-events: none; 
}
1
2
3

通过设置图片标签的鼠标事件为 none ，禁用了图片在移动端安卓的默认事件，放大预览。需要注意的是，这样设置之后图片上的点击事件也会失效，需要添加的事件绑定在外层包裹的容器上即可。
