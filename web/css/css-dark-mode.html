# css - 暗黑模式

从应用到操作系统，从苹果到谷歌，从智能手机到笔记本电脑，现在随处可见黑暗模式。宣传可以提高可见度，减少眼睛疲劳。

我们可以通过一行 CSS 快速体验简易版的暗黑模式：

html {
  filter: invert(1);
}
