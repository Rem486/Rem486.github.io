# 什么是 emoji

Emoji是一种文本类型的象形符号， 它和图片、表情包不同，它能够在任何文本输入的地方使用，因为它本身就是一种文字 ；最新的emoji 13.0版本包含1814个独立的表情符号（该统计不包含由多符号组成和其他肤色的表情符号），

emoji能够在各个不同的操作平台上显示，但显示效果会有一些不同：

emoji emoji 代码 commit 说明

🔖 (书签) :bookmark: 发行/版本标签
🐛 (bug) :bug: 修复 bug
🚑 (急救车) :ambulance: 重要补丁
🌐 (地球) :globe_with_meridians: 国际化与本地化

🚨 (警车灯) :rotating_light: 移除 linter 警告

➕ (加号) :heavy_plus_sign: 增加一个依赖
➖ (减号) :heavy_minus_sign: 减少一个依赖
⬆️ (上升箭头) :arrow_up: 升级依赖

✅ (白色复选框) :white_check_mark: 增加测试
