    # git-配置修改远程仓库地址

    创建时间：2019-02-16

    # 遇到的问题

    先在本地新建了项目，也 git init 初始化了git配置，然后才在github上新建仓库，这个时候就需要配置远程仓库的地址。

    # 解决方法

    1. 直接添加 origin 地址
    git remote add origin [url]
// git remote add origin https://github.com/Rem486/Rem486.github.io.git
    1
    2
    1. 如果已经存在了可以先删除再添加
    git remote rm origin
git remote add origin [url]
// git remote add origin https://github.com/Rem486/Rem486.github.io.git
    1
    2
    3
    1. 已经存在 origin 的，也可以直接修改
    git remote set-url origin [url]
// git remote set-url origin https://github.com/Rem486/Rem486.github.io.git
    1
    2

    tip：如果没有 origin 地址的，直接修改会提示 fatal: No such remote 'origin'
