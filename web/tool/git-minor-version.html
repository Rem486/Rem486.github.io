    # git - 快速获取小版本

    # 遇到问题

    有些仓库，内容比较多，图片版本也比较多，会导致获取完整历史的话 .git 文件比较到大，可以只 clone 最新的一个版本

    git clone 仓库地址 --depth=1

git clone 仓库地址 -b branch_name --depth=1
